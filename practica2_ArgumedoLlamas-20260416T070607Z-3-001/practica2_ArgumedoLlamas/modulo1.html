🖥️ Software de Aplicación

// Descripción del Módulo

El módulo de Software de Aplicación abarca el uso y manejo de herramientas digitales esenciales para el entorno profesional. Los estudiantes desarrollarán habilidades en la creación de documentos digitales y el manejo del desarrollo de aplicaciones.

MATERIA 1.1 📄 Documentos Digitales

Creación, edición y gestión de documentos digitales con herramientas ofimáticas avanzadas. Procesadores de texto, hojas de cálculo y presentaciones de word y excel.

MATERIA 1.2 💾 Aplicaciones Informáticas

Uso y administración de desarrollo de aplicaciones moviles. Instalación, configuración de aplicaciones.
