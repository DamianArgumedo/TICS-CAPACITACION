🎨 Software de Diseño

// Descripción del Módulo

El módulo de Software de Diseño integra creatividad y tecnología para la producción de contenido digital visual. Los estudiantes desarrollarán competencias en el desarrollo de páginas web y en herramientas profesionales de diseño digital y multimedia.

MATERIA 4.1 🌍 Páginas Web

Diseño y desarrollo de sitios web con HTML y CSS. Principios de usabilidad, accesibilidad y experiencia de usuario para la web moderna.

MATERIA 4.2 ✏️ Diseño Digital

Creación de piezas gráficas digitales con software especializado como photoshop y adobe ilustrator. Tipografía,composición visual y producción multimedia para medios digitales.
